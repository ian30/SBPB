Help

This is the simpliest app in existence. You can't do a lot here. You can do very basic HTML pages and that's about it.

Just drag an element from the tool box on the right in to the work area. Clicking on an object will allow you to change its content. You can click the SAVE button to save the changes or click the REMOVE button to remove the element from the page. click again to make the input box disappear (I'll get that fix sometime).
